Este repositório é um projeto da disciplina de Sistemas Digitais do [Departamento de Informática e Estatística](https://ine.ufsc.br/) da [Universidade Federal de Santa Catarina](https://ufsc.br/), do segundo semetre de 2025.

## Uso

Para gerar o executável `testbench`, podemos usar o comando `make`:

```sh
$ make testbench
```

Isso deve gerar um executável na pasta `build`.

Para gerar um arquivo de onda na pasta `build`, chame o comando make:

```sh
$ make simulate
```

A saída deve se chamar `simulation.ghw`.
